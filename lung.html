Lung Cancer Prediction App

PROJECT INFORMATION

  • Category: Machine Learning (Logistic Regression)
  • Project Date: July 2023
  • Web App URL: Lung Cancer Prediction App
  • Github Repository: Lung Cancer Prediction Repository
  • Libraries Used: Streamlit, Pandas, Matplotlib, Scikit-Learn, Seaborn, Imbalanced-Learn
  • Lung Cancer Survey Dataset from Kaggle
  • Utilizes Logistic Regression from the scikit-learn Library to create a Machine Learning Model
  • 25% of the dataset was used for training the model
  • Uses the Streamlit Python Library to create a Web App for real time input and predictions
  • Validation Accuracy = 98 %
