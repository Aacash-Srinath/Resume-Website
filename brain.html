Brain Tumor Detection

PROJECT INFORMATION

  • Category: Machine Learning (TensorFlow)
  • Project Date: August 2023
  • Github Repository: Github Link
  • Libraries Used: TensorFlow, Keras, Pandas, NumPy, Matplotlib
  • MRI Scan Dataset from Kaggle
  • Utilizes TensorFlow and Keras to create a Convolutional Neural Network
  • Extracts features from the MRI Scans and determines what features are used to detect for Brain Tumors, if present
  • Trained for 50 Epochs
  • Training Accuracy = 95.26 %
  • Validation Accuracy = 98 %
